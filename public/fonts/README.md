# Fonts Directory

Coloca aquí tus archivos de fuentes personalizadas.

## Formatos recomendados:
- `.woff2` (mejor compresión, soportado en navegadores modernos)
- `.woff` (fallback para navegadores más antiguos)

## Ejemplo de estructura:
```
fonts/
├── MiFuente-Regular.woff2
├── MiFuente-Bold.woff2
├── MiFuente-Italic.woff2
└── MiFuente-BoldItalic.woff2
```

## Uso:
Después de agregar tus fuentes, importalas en `src/index.css` usando `@font-face`.
